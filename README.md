# ShopEase - Ecommerce Milestone

A React + Vite ecommerce milestone website with Home, Products, Cart, and Contact pages.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This project uses `base: "/ecommerce-milestone/"` in `vite.config.js`, so the GitHub repository should be named `ecommerce-milestone` unless you change that base path.

Deploy from GitHub Actions or upload the built `dist` folder.
